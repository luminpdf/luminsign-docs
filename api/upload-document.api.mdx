Upload a document to the user's Workspace.
The document can be saved to:

- The Workspace's shared document list (`location` type: `workspace`)
- A specific Space's shared document list (`location` type: `space`)
- The user's Personal document list (`location` type: `personal`)
- A specific folder within the above location.
