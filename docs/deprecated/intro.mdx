# Lumin Sign Intro

:::warning
This section is under construction. Please check back soon.
:::

Lumin Sign provides a fully compliant digital signature flow, that can be integrated with your Web or mobile application in minutes. Let's discover **Lumin Sign in less than 5 minutes**. Lumin Sign provides functionality for:

* Send documents for signing via email
* Legally-compliant digital signatures
* Mobile and web push notifications for signing lifecycle
* Send automated or manual reminder emails for signer
* OAuth 2.0 integration

Lumin Sign is available as a standalone application, available at [luminpdf.com/lumin-sign](https://www.luminpdf.com/lumin-sign/). Lumin Sign also provides a simple integration to enable signing from your application.
